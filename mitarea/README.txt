    Instrucciones para la correcta ejecución del programa:
        - Abrir XAMPP Control Panel
        - Click en 'Start' en Apache y MySQL
        - Exportar base de datos del archivo 'prestigetravels.sql' dentro de la carpeta SQL

        - En el buscador escribir: "http://localhost/Tarea2Datos/mitarea/php/login.php"
